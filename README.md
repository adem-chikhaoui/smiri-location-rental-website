## üìã About

<img align="right" width="350" src="https://user-images.githubusercontent.com/74038190/229223263-cf2e4b07-2615-4f87-9c38-e37600f8381a.gif" alt="Development">

A **modern web-based car rental platform** that simplifies the booking process for customers and provides powerful management tools for administrators. Built with vanilla JavaScript and integrated with Google Sheets as a real-time database.

**Why Smiri Location?**

No backend required, no server costs, and instant real-time synchronization. Google Sheets provides a powerful database with built-in analytics‚Äîperfect for small to medium rental businesses.

### Installation

Clone the repository
git clone https://github.com/adem-chikhaoui/smiri-location-rental-website.git
cd smiri-location-rental-website

Set up Google Sheets (see docs/setup-guide.md)

const API_KEY = 'your-google-api-key';
